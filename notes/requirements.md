CMake - Архитектор
Зачем нужен: Чтобы организовать сборку проекта на любом ПК.
Пишешь один простой текстовый файл CMakeLists.txt, где говоришь: "У меня есть файл main.cpp, мне нужны библиотеки GLFW и GLAD, собери из этого программу". CMake смотрит на твою операционную систему, находит установленные компиляторы и автоматически создает инструкции для сборки (например, Makefile для Linux или проект Visual Studio для Windows).

GLFW - окна и клавиатура
Зачем нужен: Чтобы создать окно на экране и ловить нажатие клавиш
Библиотека сама понимает на какой ОС все работает и для нее уже выполняет нужную работу(магия)

GLAD - Переводчик видеокарты
Зачем нужен: Чтобы C++ мог отдавать команды видеокарте через OpenGL
Поскольку драйверы у всех разные и постоянно обновляются, операционная система при запуске программы не знает, по какому конкретно адресу в оперативной памяти находится нужная функция OpenGL.
Что делает GLAD:
Когда программа запускается, вызывая функцию gladLoadGLLoader(...). GLAD мгновенно пробегается по драйверу видеокарты, находит точные адреса всех сотен функций OpenGL и связывает их с C++ кодом. После этого можно спокойно вызывать любые команды рисования.
